Reconstruction de la base de donné de l’Acp

Explication mathématique de la fonction

Coordonnées des individus dans les composantes principales :

\[ \mathbf{Z} = \mathbf{X} \mathbf{P} \]

où :

  • \(\mathbf{X}\) est la matrice des données centrées et réduites,
  • \(\mathbf{P}\) est la matrice des vecteurs propres (charges).

On a déjà :

\[ \mathbf{P} = \mathbf{A} \mathbf{\Lambda}^{-1/2} \]

où :

  • \(\mathbf{A}\) est la matrice des coordonnées des variables (charges) dans le nouvel espace des composantes principales,
  • \(\mathbf{\Lambda}\) est la matrice diagonale des valeurs propres,
  • \(\mathbf{\Lambda}^{-1/2}\) représente la matrice diagonale où chaque valeur propre \(\lambda_i\) sur la diagonale est remplacée par son inverse de racine carrée \(\frac{1}{\sqrt{\lambda_i}}\).

La matrice \(\mathbf{\Lambda}^{-1/2}\) est donnée par :

\[ \mathbf{\Lambda}^{-1/2} = \text{diag}\left(\left[\frac{1}{\sqrt{\lambda_1}}, \frac{1}{\sqrt{\lambda_2}}, \ldots, \frac{1}{\sqrt{\lambda_{\text{ncp}}}}\right]\right) \]

Reconstruction des données centrées et réduites :

\[ \hat{\mathbf{X}} = \mathbf{Z} \mathbf{P}^\top \]

où :

  • \(\hat{\mathbf{X}}\) est la matrice des données reconstruites,
  • \(\mathbf{Z}\) est la matrice des coordonnées des individus,
  • \(\mathbf{P}^\top\) est la transposée de la matrice des vecteurs propres.

Retour aux unités originales :

\[ \hat{\mathbf{X}}_{\text{original}} = (\hat{\mathbf{X}} \cdot \text{ecart.type}) + \text{centre} \]

où :

  • \(\text{ecart.type}\) est un vecteur des écarts-types des variables originales,
  • \(\text{centre}\) est un vecteur des moyennes des variables originales.

Ainsi, la fonction back_to_original_data2 reconstruit les données originales en inversant les étapes de l’ACP, en utilisant les coordonnées des individus et des variables, les valeurs propres, et les paramètres de centrage et de réduction des données.

back_to_original_data2 <- function(res, ncp = NULL) {
  if (is.null(ncp)) 
    ncp <- ncol(res$ind$coord)
  ncp <- min(ncp, ncol(res$ind$coord))
  coord.ind <- as.matrix(res$ind$coord)[, 1:ncp, drop = FALSE]
  hatX <- coord.ind %*% t(sweep(as.matrix(res$var$coord)[, 1:ncp, drop = FALSE], 
                                2, sqrt(res$eig[1:ncp, 1]), FUN = "/"))
  hatX <- sweep(hatX, 2, res$call$ecart.type, FUN = "*")
  hatX <- sweep(hatX, 2, res$call$centre, FUN = "+")
  return(hatX)
}

# Reconstruction de la base de donné de l'Acp


## Explication mathématique de la fonction

**Coordonnées des individus dans les composantes principales :**

$$
\mathbf{Z} = \mathbf{X} \mathbf{P}
$$

où :

- $\mathbf{X}$ est la matrice des données centrées et réduites,
- $\mathbf{P}$ est la matrice des vecteurs propres (charges).

On a déjà :

$$
\mathbf{P} = \mathbf{A} \mathbf{\Lambda}^{-1/2}
$$

où :

- $\mathbf{A}$ est la matrice des coordonnées des variables (charges) dans le nouvel espace des composantes principales,
- $\mathbf{\Lambda}$ est la matrice diagonale des valeurs propres,
- $\mathbf{\Lambda}^{-1/2}$ représente la matrice diagonale où chaque valeur propre $\lambda_i$ sur la diagonale est remplacée par son inverse de racine carrée $\frac{1}{\sqrt{\lambda_i}}$.

La matrice $\mathbf{\Lambda}^{-1/2}$ est donnée par :

$$
\mathbf{\Lambda}^{-1/2} = \text{diag}\left(\left[\frac{1}{\sqrt{\lambda_1}}, \frac{1}{\sqrt{\lambda_2}}, \ldots, \frac{1}{\sqrt{\lambda_{\text{ncp}}}}\right]\right)
$$

**Reconstruction des données centrées et réduites :**

$$
\hat{\mathbf{X}} = \mathbf{Z} \mathbf{P}^\top
$$

où :

- $\hat{\mathbf{X}}$ est la matrice des données reconstruites,
- $\mathbf{Z}$ est la matrice des coordonnées des individus,
- $\mathbf{P}^\top$ est la transposée de la matrice des vecteurs propres.

**Retour aux unités originales :**

$$
\hat{\mathbf{X}}_{\text{original}} = (\hat{\mathbf{X}} \cdot \text{ecart.type}) + \text{centre}
$$

où :

- $\text{ecart.type}$ est un vecteur des écarts-types des variables originales,
- $\text{centre}$ est un vecteur des moyennes des variables originales.

**Ainsi, la fonction `back_to_original_data2` reconstruit les données originales en inversant les étapes de l'ACP, en utilisant les coordonnées des individus et des variables, les valeurs propres, et les paramètres de centrage et de réduction des données.**



## Progmarmation et  test de la function

```{r}
back_to_original_data2 <- function(res, ncp = NULL) {
  if (is.null(ncp)) 
    ncp <- ncol(res$ind$coord)
  ncp <- min(ncp, ncol(res$ind$coord))
  coord.ind <- as.matrix(res$ind$coord)[, 1:ncp, drop = FALSE]
  hatX <- coord.ind %*% t(sweep(as.matrix(res$var$coord)[, 1:ncp, drop = FALSE], 
                                2, sqrt(res$eig[1:ncp, 1]), FUN = "/"))
  hatX <- sweep(hatX, 2, res$call$ecart.type, FUN = "*")
  hatX <- sweep(hatX, 2, res$call$centre, FUN = "+")
  return(hatX)
}
```

```{r}
back_to_original_data2 <- function(res, ncp = NULL) {
  if (is.null(ncp)) 
    ncp <- ncol(res$ind$coord)
  ncp <- min(ncp, ncol(res$ind$coord))
  coord.ind <- as.matrix(all_previsions)
  hatX <- coord.ind %*% t(sweep(as.matrix(res$var$coord)[, 1:ncp, drop = FALSE], 
                                2, sqrt(res$eig[1:ncp, 1]), FUN = "/"))
  hatX <- sweep(hatX, 2, res$call$ecart.type, FUN = "*")
  hatX <- sweep(hatX, 2, res$call$centre, FUN = "+")
  return(hatX)
}
```
